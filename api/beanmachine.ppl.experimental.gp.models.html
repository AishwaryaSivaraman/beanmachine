class beanmachine.ppl.experimental.gp.models.SimpleGP(x_train, y_train, mean, kernel, likelihood, *args, **kwargs)

Bases: gpytorch.models.exact_gp.ExactGP, botorch.models.gpytorch.GPyTorchModel

GPytorch model that supports Bean Machine sampling and broadcasting semantics. In train mode, BM priors may be specified over GP parameters. In eval mode, this objects acts as a Gpytorch model and generates predictions using Gpytorch’s prediction strategies. For an example, see the [tutorial](link:TODO)
