class beanmachine.ppl.experimental.gp.kernels.KernelMixin(*args, **kwargs)

Bases: torch.nn.modules.module.Module

Wrapper for Gpytorch kernels that lifts parameter attributes to Bean Machine random_variable`s. This class can be freely composed with any GPytorch kernel. Any kernel with custom parameters (not defined in GPytorch) that will be inferred should extend this class and `~gpytorch.kernels.Kernel and define parameters as properties that sample from the prior and a forward() method for computing the covariance matrix:

@random_variable
def prior():
    return dist.Normal(...)

x = ScaleKernel(outputscale_prior=prior)
x.lengthscale  # returns an invocation of the prior function
               # (used during inference)
