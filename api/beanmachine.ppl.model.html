class beanmachine.ppl.model.statistical_model.StatisticalModel¶

Bases: object

Parent class to all statistical models implemented in Bean Machine.

every random variable in the model needs to be defined with function declaration accompanied with @bm.random_variable decorator.

for instance, here is Gaussian Mixture Model implementation:

K, alpha, beta, gamma = init()

@bm.random_variable def mu():

return Normal(alpha, beta)

@bm.random_variable def z(i):

return Uniform(K)

@bm.random_variable def y(i):

return Normal(mu(z(i)), gamma)
